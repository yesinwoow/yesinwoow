What is HTML?

HTML হলো web pages-এর structure তৈরির standard markup language। এটি ব্রাউজারকে নির্দেশ দেয় কীভাবে ওয়েবসাইটের কন্টেন্ট প্রদর্শন করতে হবে।

HTML History Timeline

1991

Tim Berners-Lee HTML এর প্রথম ভার্সন তৈরি করেন।

1995

HTML 2.0 রিলিজ হয়, যা প্রথম Standard HTML স্পেসিফিকেশন।

1999

HTML 4.01 পাবলিশ হয়, যা ওয়েব ডেভেলপমেন্টে একটি বড় পরিবর্তন আনে।

2014

HTML5 রিলিজ হয়, যা আধুনিক ওয়েবের foundation এবং এতে অডিও, ভিডিও সাপোর্ট যুক্ত করা হয়।

Core Elements

<h1> - <h6>

Headings

ওয়েবপেজের টাইটেল এবং সাব-টাইটেল ডিফাইন করতে ব্যবহৃত হয়।

<p>

Paragraphs

টেক্সট বা প্যারাগ্রাফ লেখার জন্য এই ট্যাগ ব্যবহার করা হয়।

<a>

Links

এক পেজ থেকে অন্য পেজে যাওয়ার জন্য হাইপারলিংক তৈরি করে।

<img>

Images

ওয়েবপেজে ছবি যুক্ত করার জন্য ব্যবহৃত হয়।

<ul>, <ol>

Lists

তথ্যগুলো বুলেট পয়েন্ট বা নাম্বারিং আকারে সাজাতে সাহায্য করে।

<form>

Forms

ইউজারের কাছ থেকে ডেটা ইনপুট নেওয়ার জন্য ফর্ম তৈরি করে।

Document Structure

<html>
Root element of the page.
<head>
Contains meta information, title, and links to CSS.
<body>
Visible page content goes here.
<h1> Hello World </h1>
<p> Paragraph content </p>
</html>

Semantic HTML

<header>
<nav>
<main>
<section>
<article>
<footer>

Why it matters? Semantic tags কোডের অর্থ ব্রাউজার এবং ডেভেলপার উভয়কেই স্পষ্টভাবে বুঝিয়ে দেয়। এটি ওয়েবসাইটের SEO উন্নত করে এবং Screen Reader ব্যবহারকারীদের জন্য Accessibility বৃদ্ধি করে।
