## 📖 About Project

A **modern, animated & aesthetic personal portfolio** designed to showcase my skills, projects, and experience — all in one place with smooth scroll animations and premium dark UI.

---

## ✨ Features

- 🌌 Animated glow orbs & grid background  
- 🧊 Glassmorphism card design  
- 🔄 Spinning gradient avatar ring  
- 📱 Fully responsive design  
- ⌨️ Scroll reveal animations  
- 🧭 Active nav link on scroll  
- 🔗 All social links in one place  
- ⚡ Zero dependencies — pure HTML/CSS/JS
