## ✨ Features

- 🎨 **Dark theme** with animated glow orbs and grid background
- 🔄 **Spinning gradient avatar ring** with smooth animation
- 📱 **Fully responsive** — works on mobile, tablet & desktop
- 🎯 **Scroll reveal animations** for all sections
- 🧭 **Active nav link** highlighting on scroll
- 💡 **Glassmorphism** card design
- ⚡ **Zero dependencies** — pure HTML/CSS/JS, no frameworks

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/yesinwoow/portfolio.git
cd portfolio
```

### 2. Add your profile photo

Place your photo inside the `images/` folder and name it `profile.png`.

### 3. Open in browser

```bash
# Simply open index.html in your browser
open index.html
```

No build tools or installation required!

## ✏️ Customization

Open `index.html` and update the following:

- **Name & title** — inside the `#home` section
- **About text** — inside `#about`
- **Projects** — add/edit `.project-card` blocks in `#projects`
- **Skills** — add/edit `.skill-box` items in `#skills`
- **Social links** — update `href` values in `#contact`
- **Profile image** — replace `images/profile.png`
